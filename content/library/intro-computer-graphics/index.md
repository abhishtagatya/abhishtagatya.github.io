The Introduction to Computer Graphics is a great introduction to deepen the knowledge of Computer Graphics for beginners to more advanced engineers. The demonstration provided using JOGL (Java OpenGL Binding) was clear enough to get the basic idea how to program something with OpenGL. However, I think that JOGL is quite outdated for its use, therefore I think this book really shines to build a foundation on what you've already learned so far in experimenting with Computer Graphics.
